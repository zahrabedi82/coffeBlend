# ☕ CoffeeBlend

A modern and responsive coffee shop website built with **HTML, CSS, and JavaScript**. Users can browse premium coffee products, add items to the shopping cart, manage quantities, and simulate the checkout process through a demo payment gateway.

## ✨ Features

- ☕ Modern coffee shop landing page
- 🛍️ Product catalog
- 🛒 Shopping cart with quantity management
- 💾 Cart persistence using Local Storage
- 💳 Demo payment modal
- 📱 Fully responsive design
- ✨ Smooth animations and interactive UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- Font Awesome
